# Introduction
NULL is a minimal, independent GNU/Linux distribution built from scratch. It is intended to be used as either a base for other distributions, or your own daily driver. It is not intended to be used as a "plug and play" distribution (yet), but rather as a foundation for you to build upon.

## Philosophy
NULL is built on the philosophy of simplicity, modularity, and user control. We believe that a distribution should be a tool, not a cage. It should be a foundation, not a finished product. It should be a canvas, not a painting.

## Installation
You can follow the [installation guide](/install) to install NULL using [pkglet](/pkglet/index).
